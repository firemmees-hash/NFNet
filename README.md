NFNET Protocol Console v1.0
Build Date: 03/15/2013

SYSTEM REQUIREMENTS:
- Python 2.7.x (optimized for 2.7.5)
- Windows XP/Vista/7 or Linux/Mac OS X
- 512MB RAM minimum
- Network connection

INSTALLATION:
1. Extract all files to a directory
2. Ensure Python 2.7 is installed
3. Run: python main.py

COMMANDS:
Type 'help' at the NFNET> prompt for available commands

FILES:
main.py              - Main program entry point
libs/                - Core libraries directory
  __init__.py        - Package initialization
  config.py          - Configuration system
  protocol.py        - Protocol implementation  
  relay.py           - Relay server
  console.py         - Command console
  client.py          - Client module
nfnet.cfg           - Configuration file (optional)

FEATURES:
- Custom NFNET protocol implementation
- Relay server for network routing
- Local intranet web interface (port 8080)
- Packet caching for performance
- JavaScript and HTML processing
- Command-line console interface
- Statistics and monitoring
